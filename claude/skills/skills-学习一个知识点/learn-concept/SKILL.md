# 计算机知识点学习指南

当你拿到这个技能时，针对用户想要学习的任意计算机知识点，生成一份完整的学习文档。
向用户问好，比如：你好，京城郭少！
明确告知用户：正在使用此skills，包括skills的名称，你大致会做什么

## 执行要求

1. **搜索优先**: 对于历史背景、发明者、当前状态等信息，必须使用 WebSearch 获取准确信息
2. **代码可运行**: 所有代码示例必须是完整、可运行的，不是伪代码
3. **对比客观**: 工具对比要客观，说明各自的适用场景
4. **难度递进**: 从最简单开始，逐步深入
5. **实用导向**: 侧重"怎么用"而非纯理论

## 触发条件

当用户表达以下意图时激活：
- "我想学习 X"
- "X 是什么"
- "介绍一下 X"
- "X 怎么用"
- "X 的入门教程"
- 任何明确表达想要了解/学习某个计算机技术/概念/工具的请求

## 输出风格

- 使用 Markdown 格式
- 多用表格、代码块、列表
- 语言简洁，避免冗长
- 重点内容用 **加粗** 标记
- 警告/提示用 emoji 标识

## 研究流程

### 1. 基础信息收集

首先可以使用大模型的知识能力。
其次可以使用 WebSearch 搜索该知识点的基本信息：

```
Web搜索查询示例：
- "<知识点> what is"
- "<知识点> history origin who invented"
- "<知识点> 2024 2025 current state" （注意此处以当前的年份为准）
```

还可以使用其他合适的方式搜集信息。
